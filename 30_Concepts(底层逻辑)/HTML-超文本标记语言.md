# 一、HTML简介

**概念**：是一种告知浏览器如何组织页面的**标记**语言。

**自述**：就是看你需要什么内容，如何用对应的标记把它标记出来（**学习HTML，就是学习标签**）。

# 二、标签

## **标签组成**：

1. 符号：标签由 <>**尖括号** 表示
2. 组成：开始标签、结束标签和内容
3. 别称：标签也称为元素

标签分类：双标签（大部分）、单标签。

## **标签**：

1. 单标签：水平线-**hr**

2. 双标签：标题-**h**（一共有1-6个、显示：字体**加大加粗**）、段落-**p**（会**自动换行**）、html - 根元素(lang用于声明网页的主要语言，en 代表英文、zh-CN  代表中文)、head - **头部元素**、title - **标题**、body - 包含了页面所有显示在**页面上的内容**、strong - **加粗**、em - *倾斜*、ins - **下划线**、del - ~~删除线~~、

**标签关系**：并列关系（兄弟关系）和嵌套关系（父子关系）

# 三、HTML文档

**文档类型**：<! DOCTYPE html> - HTML5 的文档类型声明。< meta charset="UTF-8"> - **字符集**，表示该文档的字符集设置为UTF-8。< meta name="viewport" content="width, initial-scale=1.0"> - **移动端页面适配**。
